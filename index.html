Formularios

Los formularios son elementos que permiten recopilar información del usuario del sitio web
Permite que el usuario ingrese un dato que después será utiliza por un Lenguaje de Programación.
Código HTML -> Permite indicar qué datos recopilar
Código del Lenguaje de Programación -> Permite indicar qué hacer con esos datos recopilados
PHP es un Lenguaje de Programación que se comunica con los Servidores. Para usar PHP es necesario tener un Servidor.

Elementos de Selección

En los elementos del selección el usuario responde a la pregunta ¿Querés mandar este dato?.
Si selecciona, responde que sí.
Si no selecciona, responde que no.
El atributo value se usa para indicar qué dato se manda cuando el usuario responde que sí.

input:checkbox

Elemento de selección múltiple. Permiten seleccionar uno, todos o ninguno

input:radio

Elementos de selección simple. Permite sólo la selección de uno de los elementos
